# Cypurge Mobile

Flutter starter for the Android and iOS app.

## Status

The Dart app shell is scaffolded here. The Flutter SDK is not available in this environment, so the generated native `android/` and `ios/` platform projects were not created automatically.

Once Flutter is installed, run:

```bash
flutter create --platforms=android,ios .
```

That will populate the native project files without changing the app entrypoint.
